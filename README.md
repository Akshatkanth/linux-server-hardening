Project: Linux Server Hardening

Configured a production-ready Ubuntu server on Google Cloud with:
- Non-root sudo user
- SSH key authentication (password login disabled)
- UFW firewall (SSH only)
- Fail2Ban for brute-force protection
- Automatic security updates
- Proper timezone and hostname
- Service and log management

Verified all configurations using a security checklist.
